# Evaluate Revisions

Assess whether critical concerns from the review were adequately addressed.

## Process

1. Read the review feedback (injected above) and identify all **Critical** items
2. Read the revised document (injected above)
3. For each critical item, assess: **resolved**, **partially resolved**, or **unresolved**

## Decision

Use the transition decisions below to choose the next phase.

When recommending re-review, note which specific reviewer sources still have unresolved Critical items. Only those reviewers will be re-engaged in the next review round.
